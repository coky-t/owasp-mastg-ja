## SQLite3

アプリが SQLite を使用する場合、SQLite3 ライブラリをアプリに追加する必要があります。このライブラリは SQLite コマンド用の API を提供する C++ ラッパーです。

## Couchbase Lite

[Couchbase Lite](https://github.com/couchbase/couchbase-lite-ios "Couchbase Lite") は軽量で組み込み型のドキュメント指向 (NoSQL) データベースであり、同期可能です。これは iOS および macOS 向けにネイティブにコンパイルします。
