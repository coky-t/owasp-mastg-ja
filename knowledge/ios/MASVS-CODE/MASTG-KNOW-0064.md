title: 非本番リソース (Non-Production Resources)
---

非本番リソースとは、本番使用のものではなく、開発、テスト、ステージング環境向けの資産、エンドポイント、構成のことです。これらのリソースは、セキュリティコントロールが緩かったり、デバッグ機能やテストデータを持つことが多く、意図せず本番ビルドに含まれると脆弱性をもたらす可能性があります。それらは機密情報をさらし、攻撃ベクトルを提供し、本番環境で意図しない動作につながる可能性があります。

iOS アプリでの非本番リソースのよくある例には以下があります。

- 機密情報をさらすデバッグログや詳細ログ記録フレームワーク。
- 適切な認証や暗号化がないテストサーバーや API エンドポイント。
- 偽のユーザー情報や実際のユーザー情報を含むサンプルデータファイル。
- 安全でない動作を可能にする機能フラグや構成。
- 開発用証明書やプロビジョニングプロファイル。
- テスト目的で使用されるハードコードされたクレデンシャルや API キー。
