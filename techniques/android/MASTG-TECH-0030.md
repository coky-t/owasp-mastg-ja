# MASTG-TECH-0030 サンドボックス検査 (Sandbox Inspection)

アプリケーションデータは `/data/data/<app_package_name>` にあるサンドボックス化されたディレクトリに保存されます。このディレクトリの内容については [アプリデータディレクトリへのアクセス (Accessing App Data Directories)](https://github.com/coky-t/owasp-mastg-ja/blob/master/techniques/android/MASTG-TECH-0008.md) ですでに詳しく説明しています。
