# iOS のデータストレージ

## 概要

認証トークンや個人情報などの機密データの保護はモバイルセキュリティの鍵です。この章では、ローカルデータストレージ用の iOS API と、それらを使用するためのベストプラクティスについて学びます。

永続的なローカルストレージに保存する機密データはできる限り少なくすべきです。しかし、ほとんどの実際のシナリオでは、少なくともいくつかのユーザーを保存する必要があります。幸いなことに、iOS はセキュアストレージ API を提供しており、開発者はすべての iOS デバイスで利用可能な暗号ハードウェアを使用できます。これらの API を正しく使用すると、ハードウェア基盤の 256 ビット AES 暗号化を介して機密性の高いデータやファイルを保護できます。
