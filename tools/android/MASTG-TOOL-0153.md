[dProtect](https://obfuscator.re/dprotect/) はコード難読化機能を強化した Proguard の拡張です。Proguard はシンボル (パッケージ名、クラス名、...) の難読化と Java/Kotlin コードの最適化で知られていますが、dProtect は Proguard に加えて Java と Kotlin のコード難読化を提供します。

dProtect によって提供される難読化技法には以下があります。

- 算術演算の難読化
- 定数の難読化
- 制御フローの難読化
- 文字列の暗号化
